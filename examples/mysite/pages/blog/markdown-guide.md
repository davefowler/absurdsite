# Markdown Guide

AbsurdSite uses Markdown for formatting posts. Here's a quick guide to get you started.

## Basic Formatting

- **Bold** text: `**bold**`
- *Italic* text: `*italic*`
- [Links](https://example.com): `[text](url)`

## Code Blocks

```javascript
// Code blocks use triple backticks
console.log('Hello, World!');
```
