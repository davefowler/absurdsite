# AbsurdSite

A static site generator that bakes your content into a SQLite database for offline-first performance.

## Installation

```bash
npm install
```

## Usage

Create a new site:
```bash
npm run start new my-site
```

Build the site:
```bash
npm run start build
```

Start development server:
```bash
npm run start serve
```

## Development

Build the project:
```bash
npm run build
```

Run tests:
```bash
npm test
```
